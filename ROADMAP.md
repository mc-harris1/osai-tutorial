# Roadmap

## Vision

OSaI teaches modern AI infrastructure by applying the traditional OSI networking model to a real NVIDIA DGX Spark cluster through hands-on experimentation.

---

## Phase 1 — Foundation

* Repository setup
* Core documentation
* Hardware overview
* Environment preparation
* Initial networking baseline

---

## Phase 2 — OSI Labs

* Layer 1 — Physical
* Layer 2 — Data Link
* Layer 3 — Network
* Layer 4 — Transport
* Layer 5 — Session
* Layer 6 — Presentation
* Layer 7 — Application

Each lab will include:

* Objectives
* Background
* Hands-on exercises
* Expected results
* Validation
* Discussion

---

## Phase 3 — Performance Analysis

* Throughput benchmarking
* Latency measurements
* Packet captures
* Network interface analysis
* System state comparisons
* NCCL communication experiments

---

## Phase 4 — AI Infrastructure

* Multi-node inference
* Distributed AI workloads
* NVIDIA networking stack
* ConnectX exploration
* GPUDirect technologies (where applicable)

---

## Phase 5 — Advanced Topics

* Network observability
* Performance tuning
* Scaling experiments
* Failure scenarios
* Best practices

---

## Long-Term Goals

* Additional hardware platforms
* BlueField exploration
* Ethernet vs. InfiniBand comparisons
* Community-contributed labs
* Automated benchmarking
* Interactive visualizations
